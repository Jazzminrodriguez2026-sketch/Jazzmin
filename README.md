# 📅 Planificador Semanal PWA

**Organiza tu semana con tareas, alarmas y cronómetros.** ¡Funciona como app nativa en Android e iPhone!

## 🎯 Características

✅ **Tareas semanales** - Crea tareas para cada día  
✅ **Alarmas automáticas** - Notificaciones a la hora exacta  
✅ **Cronómetro integrado** - Mide el tiempo real de cada tarea  
✅ **Historial permanente** - Registra todo lo completado  
✅ **Compartir tareas** - Envía tareas a otros usuarios  
✅ **Modo offline** - Funciona sin internet  
✅ **PWA instalable** - Se instala como app nativa  

## 📱 Instalación rápida

### iPhone (Safari)
1. Abre la app en Safari
2. Tap: **Compartir** → **Agregar a pantalla de inicio**
3. ¡Listo! Funciona como app

### Android (Chrome)
1. Abre la app en Chrome
2. Tap: **Instalar** (aparecerá automático)
3. ¡Listo! Se abre fullscreen sin navegador
